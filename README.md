# 🔄 Unit Converter Tool (CLI)

A simple Command Line Interface (CLI) based Unit Converter built using Python. This project allows users to convert values between different units such as length, weight, and temperature.

---

## 🚀 Features

- 📏 Length Conversion  
  - Kilometers ➝ Meters  
  - Meters ➝ Kilometers  

- ⚖️ Weight Conversion  
  - Kilograms ➝ Grams  
  - Grams ➝ Kilograms  

- 🌡️ Temperature Conversion  
  - Celsius ➝ Fahrenheit  
  - Fahrenheit ➝ Celsius  

- 🖥️ User-friendly CLI interface  
- ⚡ Fast and lightweight
